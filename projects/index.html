Vanlife

"Vanlife" 프로젝트는 React Router 6.4를 이용하여 만든 SPA 프로젝트임니다. 사용자의 안정적인 로그인을 다루는 과정에서 Protected Route 개념을 이용하여 개인정보를 보호하고 UX/UI를 완성시켜봤습니다.

Tenzies Game

"주사위 게임" 프로젝트는 베이직 React를 이용하여 만든 게임임니다.

My Dashboard

“My Dashboard” 프로젝트 CoinGecko API V3와 OpenWeather API를 활용하여 제작한 Chrome Extension 형태의 대시보드 애플리케이션입니다. 사용자의 Chrome 환경 설정에 따라 날씨 정보가 맞춤 적용되도록 구현하였습니다.

Movie Search 1.0

“영화 검색 V1.0” 프로젝트 The Open Movie Database(OMDb) API를 활용하여 개발한 Vanilla JS 기반 영화 검색 애플리케이션입니다.

Color Scheme

“Color Scheme” 프로젝트 Color API를 활용하여 색상 팔레트를 생성하고 조합할 수 있도록 구현한 애플리케이션입니다.

Game of War

“Game of War” 프로젝트 The Deck of Cards API를 활용하여 제작한 카드 게임 애플리케이션입니다.

Candy House

“캔디 하우스” 프로젝트 선정한 벤치마킹 사이트의 메인 페이지만 참고하고, 나머지 페이지는 자체 콘셉트에 맞게 기획 및 제작하였습니다. HTML, CSS, jQuery, JavaScript, Photoshop, Illustrator를 활용하여 구현하였습니다.

Web Profile 1.0

첫 번째 웹 프로파일 프로젝트 HTML, CSS, JavaScript를 활용하여 개인 웹 프로파일을 제작하였습니다.
